# Koa Example Server

A Postgres backed Chat example with Websockets and Koa. All enterprisy things are considered.

### Features

- [x] Koa Endpoints
- [x] Postgres
- [x] IOC
- [x] Testscontainers
- [x] Socket.io
- [x] Log4JS
- [x] Request Validation
- [x] Request Ids
- [x] Containerization
- [ ] Autogenerated API Documentation
- [ ] Dedicated Management Port
- [ ] Prometheus
- [ ] Health and Readiness
- [ ] Some Frontend
- [ ] External Apis
